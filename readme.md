# Django quiz application

## Installation
- Clone the repo
```
git clone 
```
- Change the dir and setup requirements
```
cd quiz
python3 -m venv .venv
source .venv/bin/activate # for linux or mac
.venv\Scripts\activate # for windwos
pip install -r requirements.txt
```
- Setup .env file
```
cp .env-sample .env
```
- Run the application
